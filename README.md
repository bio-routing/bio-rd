## Building

### Build the examples

#### BGP

    cd examples/bgp/ && go build

#### BMP

    cd examples/bmp/ && go build

#### Device

    cd examples/device && go build

### Run Tests

    go test -v -cover ./...

### Update modules

    go mod tidy
